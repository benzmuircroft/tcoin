# vcoin
a digital coin use NodeJS
## Install
  * Install NodeJS. [Download](https://nodejs.org)
  * Open command line (on Windows) or terminal (on Linux, MacOS):
  `npm install -g tcoin`
  * After install. Run
  `node app.js`
## Use:
  Open link: [http://localhost:3000](http://localhost:3000)
